# 09 — Exemplos de CLI

## Scan

```bash
kryonix home scan
```

## Relatório

```bash
kryonix home report
```

## Duplicatas

```bash
kryonix home duplicates
```

## Plano dry-run

```bash
kryonix home plan --dry-run
```

## Plano em JSON

```bash
kryonix home plan --json | jq .
```

## Futuro: análise IA

```bash
kryonix home analyze --ai --limit 20
```

## Futuro: busca

```bash
kryonix home search "comprovante de pagamento da internet"
```

## Futuro: explicar arquivo

```bash
kryonix home explain ~/Documentos/Financeiro/Comprovantes/arquivo.pdf
```

## Futuro: rollback

```bash
kryonix home rollback 20260508-120000-inspiron
```
