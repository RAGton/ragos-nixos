# Agent Rules

Este diretório contém as políticas obrigatórias para agentes operando no repositório Kryonix.

## Estrutura
- **00-core.md:** Regras fundamentais e filosofia de operação.
- **10-documentation.md:** Padrões de escrita e manutenção de docs.
- **20-testing.md:** Protocolos de validação e testes.
- **30-nixos.md:** Regras específicas para o ecossistema NixOS.
- **40-security.md:** Políticas de segurança e proteção de dados.
- **50-rag-brain.md:** Governança sobre o sistema de RAG e Brain.
- **90-definition-of-done.md:** Critérios finais para entrega de tarefas.

## Como usar
Todo agente deve ler estes arquivos antes de realizar qualquer alteração significativa. Eles servem como "guardrails" para garantir que o desenvolvimento seja controlado, seguro e testável.
