# Definition of Done (DoD)

Uma tarefa só pode ser considerada pronta se satisfizer os seguintes critérios:

- [ ] **Implementação:** A alteração foi feita conforme o plano.
- [ ] **Teste:** A funcionalidade foi testada e não quebrou componentes existentes.
- [ ] **Evidência:** Os resultados dos testes foram registrados (ex: em `docs/TESTING.md`).
- [ ] **Documentação:** A documentação técnica (`docs/`) foi atualizada para refletir a mudança.
- [ ] **Audit:** O script `scripts/doc-audit.sh` passou sem erros fatais.
- [ ] **Riscos:** Eventuais riscos residuais foram listados e comunicados.
