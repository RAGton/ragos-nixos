# Kryonix Brain Antigravity Agent Pack

Pacote para orientar o Antigravity a estabilizar e evoluir o `kryonix-brain-lightrag`.

Objetivo:
- Corrigir defeitos atuais do LightRAG/Kryonix Brain.
- Implementar CAG completo como camada rápida de contexto canônico.
- Melhorar respostas, grounding, links do grafo, Obsidian, categorias e cores.
- Validar NixOS/Glacier sem quebrar boot, SSH, GPU, rede ou storage.

Ordem de uso:
1. Cole o conteúdo de `AGENTE.md` no Antigravity.
2. Anexe/aponte este pacote como referência do trabalho.
3. Exija que ele leia `rules/`, `skills/`, `workflows/` e `checklists/definition_of_done.md`.
4. Não autorize `nh os switch` antes de build/test/rollback documentados.
