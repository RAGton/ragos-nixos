# Definition of Done

## Bugfix

- [ ] MCP `rag_search` não quebra com dict.
- [ ] `httpx` declarado.
- [ ] API falha fechada sem chave.
- [ ] `--no-cache` aceito e funcional.
- [ ] subprocess propaga erro.
- [ ] logger corrigido.
- [ ] versão LightRAG pinada ou faixa segura.

## CAG

- [ ] `rag cag build`.
- [ ] `rag cag status`.
- [ ] `rag cag ask`.
- [ ] `rag cag route`.
- [ ] manifest com hash/fontes/tokens.
- [ ] fallback para RAG se CAG estiver stale.

## Grafo

- [ ] entidades categorizadas.
- [ ] cores/frontmatter.
- [ ] MOCs gerados.
- [ ] links Obsidian melhores.
- [ ] graph report criado.

## Qualidade

- [ ] resposta tem schema estável.
- [ ] fontes aparecem quando há grounding.
- [ ] low confidence bloqueia resposta inventada.
- [ ] eval suite inicial existe.
- [ ] perguntas críticas não alucinam.

## NixOS

- [ ] paths parametrizados.
- [ ] API bind em 127.0.0.1 por padrão.
- [ ] warmup de modelo desligado por padrão.
- [ ] `nix flake check` passa.
- [ ] `nh os build .#glacier` passa.
- [ ] não foi rodado switch sem aprovação.

## Relatório final

- [ ] arquivos alterados.
- [ ] comandos rodados.
- [ ] resultados.
- [ ] riscos.
- [ ] rollback.
