# Project Context

Este diretório contém o estado operacional temporário do projeto, otimizado para o consumo por agentes de IA.

## Arquivos
- **CURRENT_STATE.md:** Resumo do estado atual do sistema e serviços.
- **REPO_MAP.md:** Mapa simplificado da estrutura de diretórios e responsabilidades.
- **CONSTRAINTS.md:** Restrições técnicas e limitações do ambiente atual.
- **ACTIVE_WORK.md:** Registro da tarefa em andamento e progresso.

## Regras de Manutenção
1. **Contexto Curto:** Mantenha os arquivos objetivos e sem histórico longo.
2. **Sem Segredos:** Nunca coloque dados sensíveis aqui.
3. **Ponte para Docs:** Use links para as documentações canônicas em `docs/`.
