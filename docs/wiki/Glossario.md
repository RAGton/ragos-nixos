# Glossário

Status: Implementado (referência)

## Resumo
Definições rápidas dos termos mais usados no Kryonix.

## Termos
- **Kryonix:** plataforma NixOS declarativa do projeto.
- **Glacier:** host servidor IA e workstation gamer.
- **Inspiron:** host cliente/workstation.
- **Brain:** central de IA (LightRAG + MCP + Vault + Ollama).
- **LightRAG:** motor GraphRAG do Kryonix.
- **CAG:** cache de contexto (roadmap).
- **MCP:** Model Context Protocol (JSON-RPC para agentes).
- **Vault:** Obsidian vault com conhecimento técnico.
- **Flake:** entrada declarativa NixOS/outputs do projeto.
- **Profile:** composição reutilizável por papel.
- **Feature:** capacidade opt-in do sistema.

## Quando usar
Para alinhar termos antes de ler a documentação técnica.

## Comandos relevantes
```sh
kryonix doctor
```

## Riscos
- Confundir cliente/servidor ao interpretar o vocabulário.
