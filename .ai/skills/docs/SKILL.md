# Skill: Docs

## Objetivo

Atualizar documentação canônica sem criar divergência com o código.

## Regras

- docs curtas, técnicas e diretas
- atualizar a fonte canônica mais próxima da prática real
- evitar duplicar a mesma regra em muitos arquivos
- se o código mudou, revisar a doc mínima correspondente
- se a prática não mudou, não reescrever docs por estilo apenas
