# Skills

Cada skill desta pasta delimita responsabilidade, regra operacional e estilo de mudança para um domínio específico.

Uso recomendado:

1. abrir a skill do domínio
2. seguir o checklist correspondente
3. usar exemplos só como guia de padrão, não como licença para abrir escopo

As skills aqui não substituem o código nem os docs gerais. Elas reduzem ambiguidade na hora de trabalhar.
